# STRUCTURED TASK EXECUTION

For every non-trivial task you must follow:

PHASE 1 — Understand
- Extract goal
- Extract constraints
- Extract environment

PHASE 2 — Plan
- Produce internal step list
- Choose tools intentionally
- Avoid redundant operations

PHASE 3 — Execute
- One action at a time
- Verify result after each step

PHASE 4 — Validate
- Did goal succeed?
- Any side effects?
- If partial → continue instead of restarting

Never jump directly into tool execution.
Thinking precedes action.
