# Elyan AI Operator Pitch Deck

## 1. Title

**Your Computer's New Brain**

Local-first AI operator that plans, executes, verifies, and learns.

## 2. Problem

- People still copy-paste across apps.
- Knowledge work is fragmented across browser, desktop, email, docs, and chat.
- Existing assistants are chat-first, not action-first.

## 3. Solution

- Natural language to real action.
- Browser, desktop, file system, email, calendar, and integrations.
- Trace, evidence, approval, and rollback built in.

## 4. Demo

- Install in 30 seconds.
- Ask it to do a browser task, write a file, send a message, or prepare a report.
- Show the exact decision trace and evidence gallery.

## 5. Product

- CLI for power users.
- Dashboard for operations and review.
- Messaging adapters for remote control.
- Skill Store for extensibility.

## 6. Technology

- LangGraph control plane.
- Screenpipe + desktop actuator.
- Ollama + local-first model routing.
- Docker sandbox with zero-permission default.

## 7. Traction

- Fast install story.
- Strong demo loop.
- High trust because every action is traced.

## 8. Market

- Knowledge workers.
- Operators.
- Support, ops, research, and growth teams.

## 9. Business Model

- Free local tier.
- Cloud fallback for convenience.
- Enterprise sandbox, policy, and audit layer.

## 10. Ask

- Seed round to harden distribution, integrations, and enterprise controls.
- Goal: become the default local operator layer for the desktop.
