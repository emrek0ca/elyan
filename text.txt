Giriş: Ne Bu Asistan?
Bu açık kaynaklı proje, kendi bilgisayarınızda çalışan, özerk bir yapay zeka asistanıdır. Tercih ettiğiniz sohbet uygulamalarından (WhatsApp, Telegram, Discord, Slack, Signal, iMessage vb.) erişebileceğiniz ve sizin adınıza gerçek işlemler gerçekleştirebilen bir sistem sunmaktadır.

Kurulum Aşamaları
1. Sistem Gereksinimleri

Node.js 22+ (daha eski sürümler desteklenmiyor)
macOS, Linux veya Windows (WSL2 üzerinde)
2GB RAM (temel sohbet) veya 4GB RAM (web taraması için)
İnternet bağlantısı

2. Kurulum Yöntemi: Tek Komut (Önerilen)
macOS/Linux:
bashcurl -fsSL https://elyan.ai/install.sh | bash
Windows (PowerShell):
powershelliwr -useb https://elyan.ai/install.ps1 | iex
Bu komut otomatik olarak:

Node.js'i kontrol eder/yükler
CLI'ı genel olarak kurur (npm aracılığıyla)
Onboarding sihirbazını başlatır

3. Onboarding Sihirbazı Çalıştırma
bashelyan onboard --install-daemon
Sihirbaz adım adım şunları yapılandıracaktır:

AI sağlayıcı seçimi (Anthropic, OpenAI, Google veya yerel model)
API anahtarı konfigürasyonu
Sohbet kanalı bağlantısı (Telegram, WhatsApp vb.)
Asistan kişiliği ve davranışı
Oto-başlatma ayarları (launchd/systemd)

4. Gateway Başlatma ve Kontrol
bash# Gateway'i başlat
elyan gateway start

# Durum kontrol et
elyan gateway status

# Logları görüntüle
elyan gateway logs

# Yeniden başlat
elyan gateway restart
5. Web Arayüzü (Control UI)
bashelyan dashboard
Tarayıcı otomatik olarak açılır: http://127.0.0.1:18789/

Temel Yetenekler (Tools)
Sistem şu altı temel araç grubunu içerir:
1. Dosya Sistemi (group:fs)

read - Dosya ve klasörleri oku
write - Yeni dosya oluştur/var olana yazıyor
edit - Dosya içeriğini düzenle
apply_patch - Çok parçalı düzeltmeler uygula

2. Çalıştırma ve Bash (group:runtime)

exec - Shell komutlarını çalıştır
bash - Bash script'leri çalıştır
process - Sistem işlemlerini yönet

3. Web Araçları (group:web)

web_search - İnternet arama yap (Google benzeri)
web_fetch - URL'den HTML/metin al (45.000+ karaktere kadar)

4. Tarayıcı Kontrol (group:ui)

Dedike bir Chrome/Chromium örneğini kontrol et
Sayfalara tıkla, form doldur, ekran görüntüsü al
Chrome DevTools Protocol (CDP) aracılığıyla hızlı işlem
Çoklu tarayıcı profili desteği (100 profile kadar)

5. Sohbet ve Mesajlaşma (group:messaging)

Bağlı tüm kanallara mesaj gönder
Tepki (reaction) ekle, mesajları düzenle/sil
Diğer kanallardaki eylemleri tetikle (Discord/Slack)

6. Otomasyon (group:automation)

cron - Görevleri zamanla (her saat, günde vb.)
gateway - Gateway olaylarını tetikle
Heartbeat: Düzenli aralıklarla (ör. her 6 saat) uyandır

7. Bellek (group:memory)

Kullanıcı tercihlerini hatırla (24/7 kalıcı)
Geçmiş bağlam sakla (otomatik özetleme)
Arama işlevselliği

bash# Tarayıcıyı kontrol et

elyan browser type [text]      # Metin yaz

PDF indir ve işle

3. Cron İşleri - "Uyandığında Yapacağı Görevler"

elyan cron add --expression "0 18 * * *" \
  --prompt "Trending konuları Reddit/HN'den topla, Telegram'a gönder"

7. Multi-Agent Routing
Gelişmiş kurulum:
json{
  "agents": [
    {
      "id": "work",
      "workspace": "./work-workspace",
      "routes": ["slack", "github"]
    },
    {
      "id": "personal",
      "workspace": "./personal-workspace",
      "routes": ["telegram", "whatsapp"]
    }
  ]
}

Yapılandırma Dosyası (elyan.json)
Yapılandırma ~/.elyan/elyan.json dosyasında JSON5 formatında:
json5{
  // AI Modeli seçimi
  models: {
    default: {
      provider: "anthropic",  // anthropic, openai, google, local
      model: "claude-opus-4-5-20251101",
      apiKey: "$ANTHROPIC_API_KEY"
    }
  },

  // Tools'ı kısıtla (güvenlik)
  tools: {
    allow: ["group:fs", "group:web", "browser"],
    deny: ["exec"]  // Komut çalıştırmayı engelle
  },

  // Sandbox (Docker) ayarları
  sandbox: {
    enabled: true,
    mode: "full"  // docker, host, atau restricted
  },

  // Kanal yapılandırması
  channels: [
    {
      type: "telegram",
      token: "$TELEGRAM_BOT_TOKEN",
      enabled: true
    }
  ],

  // Cron görevleri
  cron: [
    {
      expression: "0 6 * * *",
      prompt: "Daily briefing"
    }
  ],

  // Hareket frekansu (heartbeat)
  heartbeat: {
    enabled: true,
    intervalMinutes: 360  // Her 6 saatte bir uyandır
  }
}
```

## Gerçek Dünya Kullanım Örnekleri

### **Örnek 1: Sabah Brifing Otomasyonu**
```
Saat 6:47'de otomatik olarak:
✓ Günün takvimini kontrol et
✓ Bekleyen e-postaları listele
✓ Hava durumunu al
✓ CI/CD başarısızlıklarını kontrol et
→ Telegram'a gönder
```

### **Örnek 2: E-posta Triyajı**
```
Günde iki kez:
✓ Tüm e-postaları tara
✓ Kategorize et (acil, standart, spam)
✓ Newsletter'ları arşivle
✓ İşlem gerekenleri işaretle
→ Özeti gönder
```

### **Örnek 3: Web Taraması Otomasyonu**
```
"Trending konuları topla" komutu:
✓ Reddit/Hacker News'i tara
✓ Yazı konularına ilişkin tartışmaları ayıkla
✓ Markdown dosyasına kaydet
✓ Slack'te paylaş
```

### **Örnek 4: Kod Devops İş Akışı**
```
GitHub repo'da hata algılandığında:
✓ Logları al (Sentry webhook)
✓ Sorunu analiz et
✓ Test'i çalıştır
✓ PR oluştur
✓ Discord'da bildir
